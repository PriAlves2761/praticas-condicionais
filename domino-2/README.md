
## Dando nome as pedras 

Modifique o código anterior para que, quando a pedra for uma bucha, digamos qual o "nome" da bucha. Para quem não constuma jogar dominó, segue um glossário:

| Bucha de | Nome   |
| -------- | ------ |
| 0        | Branco |
| 1        | Ás     |
| 2        | Duque  |
| 3        | Terno  |
| 4        | Quadra |
| 5        | Quina  |
| 6        | Sena   |
